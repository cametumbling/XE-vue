# XE-vue

A project in two parts:
- Tabbed browsing of 3 cities (dynamic "selected" class yet to be added)
- City weather search

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
